Recommandations

  • ✓ Taille minimum 16px pour texte courant
  • ✓ Line-height 1.5-1.6 pour espacement vertical
  • ✓ Padding généreux (easier to click)
  • ✓ Ombres douces (crée profondeur)
  • ✓ Transitions 200-300ms
  • ✓ Pas de jargon technique
